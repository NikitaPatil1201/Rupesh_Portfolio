# Personal Portfolio Website

A modern, elegant, and fully responsive personal portfolio website built using **HTML, CSS, and JavaScript** with smooth animations, dark/light mode toggle, mobile responsiveness, and a clean professional layout.

This portfolio is designed to showcase your **projects, resume, skills, experience, and contact details** in a premium and minimal style.

---

## ✨ Features

- Fully Responsive Design
- Dark / Light Theme Toggle (saved using localStorage)
- Smooth Scroll Navigation
- Modern Hero Section
- About / Summary Section
- Projects Showcase Cards
- Resume & Experience Timeline
- Skills Section with Categories
- Contact Form (Netlify Ready)
- Mobile Navigation Menu
- Scroll Reveal Animations
- Elegant Typography using Google Fonts
- Clean UI with Professional Layout

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **Google Fonts**
- **Netlify Form Support**
